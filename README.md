# Hello Expo App (Astig na App) 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Description.
This app contains a text input which will greet you once you input your name. It also has a built-in number counter. It features Vergil from the Devil may Cry series.

## How tu run.

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

3. Scan the QR code using Expo Go.
